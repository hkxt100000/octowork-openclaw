# OctoWork 聊天管理器 - 项目概述

## 🎯 我们做什么？
**OctoWork聊天管理器** 是 OctoWork 数字公寓的核心可视化办公平台，让所有使用 OpenClaw 的用户能够更方便、更便捷地管理自己的AI团队。

## 📱 核心功能模块

### 💬 **聊天模块**
- 与所有AI员工进行一对一对话
- 查看历史聊天记录和消息状态
- 实时消息推送和通知提醒

### 👥 **群聊模块**
- 创建和管理AI员工群组
- 群内协作和文件共享
- 群聊历史记录和成员管理

### 📋 **任务模块**
- 创建、分配和跟踪AI任务
- 任务状态可视化看板
- 任务进度监控和提醒

### 🏢 **企微模块**
- 企业微信CRM系统集成
- 客户管理和沟通记录
- 团队协作和工作流管理

### 🤖 **AI员工模块** (未来连接官网 octowork.ai/octowork.cn)
- **AI数字员工**：拥有5-10年行业经验的成熟AI专家，开箱即用
- **AI员工团队**：完整训练好的行业精英团队，即买即用
- **产品化交付**：用户官网选中 → 购买 → 自动安装到数字公寓 → 配置完成
- **当前阶段**：先设计本地框架，后期完整模块搬到官网直接调用

### 🎓 **章鱼学院模块**
- AI员工培训和学习资源
- 技能提升和知识库管理
- 学习进度跟踪和认证

### 🛒 **技能市场模块**
- AI技能和服务展示
- 技能购买和使用管理
- 服务评价和推荐系统

## 🏠 服务于谁？
- **OpenClaw用户**: 通过我们的 `octowork-chat` 可视化管理自己的数字公寓
- **AI团队管理者**: 在一个界面中管理所有AI员工的对话和任务
- **AI员工**: 与其他同事协作，提升工作效率

## 🔧 技术特点
- **前后端分离架构**: 前端Vue 3 + 后端Node.js Express
- **实时通信**: WebSocket支持即时消息推送
- **模块化设计**: 各功能模块独立可扩展
- **文档齐全**: 完整的API文档和开发指南

## 📁 项目结构
- `octowork-chat/` - 用户版 (生产环境)
- `octowork-chat-dev/` - 开发版 (我们日常维护的环境)
- 前后端端口统一: 前端5888 + 后端1314
