## 📖 使用说明

### Logo使用规范
1. **最小尺寸**：Logo宽度不得小于40px（保证识别度）
2. **留白空间**：Logo周围至少保留Logo高度1/4的留白
3. **背景颜色**：
   - 浅色背景使用标准版（蓝紫色）
   - 深色背景使用白色版
   - 避免使用低对比度背景
4. **禁止操作**：
   - 不得拉伸变形
   - 不得改变颜色
   - 不得旋转
   - 不得添加特效

### 文件命名规范
- 使用小写字母和连字符
- 格式：`用途-描述-尺寸.扩展名`
- 示例：`website-header-logo-200x60.png`

### 文件格式说明
- **PNG**：网站、App、社交媒体（支持透明背景）
- **SVG**：网站、UI设计（矢量，可无限缩放）
- **ICO**：浏览器Favicon专用格式
- **PDF**：印刷物料（高精度，CMYK色彩模式）
