## 🔧 技术架构

### 后端
- **框架**: Node.js + Express
- **端口**: 6726
- **API**: RESTful + WebSocket
- **数据库**: SQLite
- **状态**: ✅ 正常运行

### 前端
- **框架**: Vue 3 + Vite
- **端口**: 5174
- **UI**: Element Plus
- **路由**: Vue Router
- **状态**: ✅ 正常运行

### API端点
| 端点 | 方法 | 状态 | 说明 |
|------|------|------|------|
| /api/auth/login | POST | ✅ | 登录 |
| /api/auth/me | GET | ✅ | 当前用户 |
| /api/bots | GET | ✅ | Bot列表 (58个) |
| /api/teams | GET | ✅ | Team列表 (9个) |
| /api/groups | GET | ✅ | 群组列表 |
| /api/tasks | GET | ✅ | 任务列表 |
| /api/bots/:id/messages | GET | ⏳ | Bot消息 |
| /api/bots/:id/message | POST | ⏳ | 发送消息 |
