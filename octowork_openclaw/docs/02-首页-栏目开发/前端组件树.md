## Login.vue — 登录页 (1078行)

### 模板结构 (五层)

```
<div class="login-page">
  层1: .video-bg          ← 视频背景轮播 + 音量控制按钮
  层2: header.top-nav     ← Logo + 功能大全下拉 + 导航链接 + 登录按钮
  层3: .center-content    ← 主 Slogan + 进化输入框 (带文案轮播)
  层4: .video-switcher    ← 右侧透明滑条 (鼠标滚轮/点击切换视频)
  层5: transition modal   ← 登录弹窗 (Element Plus 表单)
</div>
```

### 核心状态

| 变量 | 类型 | 说明 |
|------|------|------|
| `currentIndex` | ref(0) | 当前视频索引 |
| `isMuted` | ref | 静音状态 (持久化 localStorage) |
| `showLoginModal` | ref(false) | 登录弹窗可见性 |
| `loginForm` | reactive | `{username, password, remember}` |
| `showFeaturesMenu` | ref(false) | 功能大全下拉菜单 |
| `subtitleIndex` | ref(0) | 当前文案轮播索引 |

### 核心逻辑

| 功能 | 实现 |
|------|------|
| 视频轮播 | `VIDEO_LIST` 配置驱动，`onended`/`ontimeupdate` 双重检测，备用 setTimeout |
| 文案轮播 | `videoSubtitleSets` 对象，每 5 秒切换，随视频切换重置 |
| 音量控制 | `isMuted` ref + localStorage 持久化 |
| 登录提交 | `authService.login()` → 成功后 `router.push('/')` |
| 鼠标滚轮切换 | `@wheel.prevent="handleWheel"` → 防抖切换视频 |

### 依赖

- `config/videoConfig.ts` — VIDEO_LIST, VIDEO_DURATION, VIDEO_TRANSITION
- `services/auth.ts` — AuthService 单例 (login/logout/isAuthenticated)

## services/auth.ts — 认证服务 (237行)

```typescript
class AuthService {
  // 状态: token + currentUser (localStorage 持久化)

  login(username, password, remember) → POST /api/auth/login → 存 token + user
  getCurrentUser()                    → GET /api/auth/me
  updateUser(updates)                 → PUT /api/auth/update
  changePassword(old, new)            → POST /api/auth/change-password
  logout()                            → 清除 localStorage + axios header
  isAuthenticated()                   → boolean (token && currentUser 均存在)
  getToken()                          → string | null
  getCachedUser()                     → User | null (不发请求)
}

export const authService = new AuthService()  // 单例
```

### User 类型

```typescript
interface User {
  userId: string
  username: string
  nickname?: string
  avatar?: string
  workspace_code?: string
  email?: string
  phone?: string
  role?: string
  settings?: { theme, language, notifications }
}
```
