## 一、群聊功能增强

### 1.1 群聊列表 — 最新消息 & 未读数

| 改动位置 | 文件 | 说明 |
|----------|------|------|
| composable | `frontend/src/renderer/composables/useGroupChat.ts` | `groupLastMessages` 每次 `loadGroupsLocal()` 均从 API 刷新；发送消息后立即更新 |
| 侧边栏 | `ContentSidebar.vue` | 每个群组卡片显示 `last_message` 摘要 + 每群未读红标 |
| 全局导航 | `MainLayout.vue` | 导航栏"群聊"图标右上角显示 `unreadGroupCount` 总数红标 |

**后端接口**:
```
GET /api/groups
→ 返回 { success: true, groups: [...] }
  每个 group 对象新增 last_message: { sender, sender_name, content, timestamp } | null
```

### 1.2 群聊打开时自动滚动到底部

| 改动位置 | 文件 | 说明 |
|----------|------|------|
| composable | `useGroupChat.ts` → `selectGroup()` | 等待 `loadGroupMessagesLocal` + `nextTick()` 后调用 `scrollToBottom()` |
| 群聊视图 | `GroupChatView.vue` | 新增 watcher：`selectedGroup` 变化后 100ms 延迟滚底 |
| 消息视图 | `GroupMessagesView.vue` | 新增 watcher：`groupMessages.length` 变化后 150ms 延迟滚底（兼容图片异步加载） |

### 1.3 消息保留策略 (40-45-5)

为防止 `sessionStorage` 膨胀，当 API 返回消息数 >= 45 条时，前端只保留最新 5 条。

| 触发位置 | 文件 | 逻辑 |
|----------|------|------|
| 加载消息 | `useGroupChat.ts` → `loadGroupMessagesLocal()` | `if (msgs.length >= 45) msgs = msgs.slice(-5)` |
| 实时消息 | `useGroupChat.ts` → `handleGroupNewMessage()` | 同上规则裁剪 `allGroupMessages` |

### 1.4 实时聊天记录文档 (Markdown)

后端首次收到群聊消息时自动创建 `.md` 聊天记录文件。

| 改动位置 | 文件 | 说明 |
|----------|------|------|
| 后端控制器 | `backend/src/controllers/groupController.js` → `appendToGroupChatHistory()` | 路径: `departments/<deptId>/chat_history/YYYY-MM-DD.md` |
| 文件格式 | 同上 | 头部含日期、群名、消息计数表格；消息区新→旧排列 |
| 用户消息 | blockquote `>` 引用 + `HH:MM:SS` 时间戳 | 示例: `> 内容文字` |
| Bot 消息 | 普通段落文本 | 无引用前缀 |

**后端接口**:
```
GET /api/groups/:groupId/chat-records
→ { success: true, records: [...], count: N }

GET /api/groups/:groupId/chat-records/:fileName
→ { success: true, content: "..." }
```

## 三、接口速查总览 (含本次更新)

### 群聊接口

| 方法 | 路径 | 说明 | 本次新增/更新 |
|------|------|------|--------------|
| GET | `/api/groups` | 获取所有群组 (含 `last_message`) | 更新 |
| POST | `/api/groups/:groupId/messages` | 发送群聊消息 | - |
| GET | `/api/groups/:groupId/messages?limit=50` | 获取群聊消息 | - |
| DELETE | `/api/groups/:groupId/messages` | 清空群聊消息 | - |
| POST | `/api/bot/send-to-group` | Bot 主动发群消息 | - |
| GET | `/api/groups/:groupId/chat-records` | 获取群聊聊天记录列表 | 更新 |
| GET | `/api/groups/:groupId/chat-records/:fileName` | 获取某条聊天记录内容 | 更新 |
| GET | `/api/departments/:deptName/chat-records` | 获取部门聊天记录 | - |

### 单聊接口 (无变化，仅修复)

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/messages/:botId?limit=50&offset=0` | 获取消息历史 |
| POST | `/api/messages/:botId` | 发送消息 |
| DELETE | `/api/messages/:botId` | 清空消息 |
| GET | `/api/sessions` | 获取最近会话 |
| GET | `/api/bots/:botId/file-tree` | 获取文件树 |
| GET | `/api/bots/:botId/chat-records` | 获取 Bot 聊天记录 |

### WebSocket 消息格式 (含本次更新)

```javascript
// 群聊新消息推送
{
  type: 'group_message',
  groupId: string,
  message: {
    id: number,
    sender: string,      // 'user' | botId
    sender_name: string,
    content: string,
    timestamp: number,
    mentions?: string[]
  }
}

// 群聊通知
{
  type: 'group_notification',
  groupId: string,
  notification: { ... }
}

// 任务预览 (群聊检测到任务意图时)
{
  type: 'task_preview',
  groupId: string,
  task: {
    title: string,
    assignee: string,
    priority: string,
    deadline: string,
    confirmToken: string
  }
}
```
